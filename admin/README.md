# Membership Auto - Admin Dashboard

A comprehensive admin dashboard built with Next.js 15 for managing the Membership Auto platform.

## Features

- **Dashboard Overview**: Real-time KPIs, revenue charts, membership distribution
- **Member Management**: View, search, edit members and subscriptions
- **Vehicle Management**: Track all vehicles, health status, maintenance
- **Appointment Management**: Schedule, assign, track service appointments
- **Offers CMS**: Create and manage promotional offers
- **Chat Management**: Handle customer support conversations
- **Referral Tracking**: Monitor referral program performance
- **Location Management**: Manage service locations and capacity

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

Login with Django admin credentials (requires `is_staff=True`)

## Tech Stack

- Next.js 15 + React 19
- TypeScript + Tailwind CSS 4
- Recharts for analytics
- Token-based auth with Django backend
