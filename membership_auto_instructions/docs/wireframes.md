Wireframes & Screen Specs (Mobile-first)
======================================

Follow these screen specs to build Figma frames.

1. Splash
- Center logo, tagline. Auto-redirect.

2. Auth: Login / Signup / Forgot
- Email, password, phone, referral code
- Social sign-in optional

3. Main Tabs (bottom):
- Dashboard, Vehicles, Appointments, Offers, Account

4. Dashboard
- Top: membership card (plan, renewal)
- Middle: active vehicle card (make/model/year, mpg preview)
- Grid: golden icons (6-8), each with label
- Bottom: recent activity feed

5. Vehicles List
- + Add Vehicle button
- Each row: make/model year | plate | health dot
- Tap -> Vehicle Detail

6. Vehicle Detail
- Health snapshot (DTC count, last service)
- Fuel charts (30d)
- Pair/Unpair dongle button
- Service history list

7. Fuel Add Flow (manual)
- Photo (receipt), Odometer input, Gallons, Price
- AI preview -> confirm

8. Appointment Booking Flow
- Choose vehicle -> choose location -> pick time slot -> confirm

9. Chat
- Thread list -> messages -> attachments

Provide these frames to designer; maintain 8pt grid and corner radius 12.
